# HProjects - Pagina de Login

<img width="1918" height="910" alt="Captura de tela 2026-03-18 220254" src="https://github.com/user-attachments/assets/61728a66-4e9e-4afa-84f7-53aeacc7161d" />

Pagina de login responsiva construida com React e Vite, com ilustracao espacial decorativa e formulario de autenticacao.

---

## Funcionalidades

- Layout em dois paineis: ilustracao a esquerda e formulario a direita
- Validacao de email e senha no envio do formulario
- Checkbox de "Lembrar de mim"
- Links de "Esqueceu a senha" e "Crie uma conta" (placeholders)
- Design responsivo para dispositivos moveis e tablets
- Tema roxo escuro com variaveis CSS
---

## Regras de Validacao

- Os dois campos sao obrigatorios
- O email deve estar em formato valido (`usuario@dominio.com`)
- Erros sao exibidos em linha acima do botao de envio
